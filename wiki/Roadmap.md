# Roadmap

This page tracks medium-term product direction. Keep it updated each sprint.

## Planned

- Apparent solar time mode (Equation of Time)
- Sunrise and sunset indicators
- Additional premium skins and motion polish
- Expanded widget customization and lock screen support

## Candidate Enhancements

- Better onboarding for permission rationale
- Regional presets for frequently used longitudes
- Export/share current solar offset view

## Process

- Move items from Planned to In Progress when implementation starts.
- Link each in-progress item to an issue or milestone.
- Mark shipped items with release version.
