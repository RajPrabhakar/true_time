# TruTime Wiki

Welcome to the project wiki for TruTime.

TruTime is a Flutter app that shows local solar time (Local Mean Time) from device longitude:

LMT = UTC + (longitude x 4 minutes)

Use this wiki for architecture notes, development workflows, testing, and roadmap planning.

## Current Status

- Core app is implemented for Flutter mobile and desktop targets.
- Solar time calculation is offline and on-device.
- Theme system supports free, premium, and skin categories.
- Unit and golden tests are included.
